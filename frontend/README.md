# BadNews Frontend

Frontend Vue 3 + Vite para compradores.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Estructura

- `src/components/` - Componentes Vue reutilizables
- `src/pages/` - Páginas principales
- `src/services/` - Servicios API
- `src/stores/` - Almacenamiento global (Pinia)
- `src/router.js` - Configuración de rutas
