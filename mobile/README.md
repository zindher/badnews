# BadNews Mobile App

App Flutter para Mensajeros.

## Setup

```bash
flutter pub get
flutter run
```

## Estructura

- `lib/models/` - Modelos de datos
- `lib/screens/` - Pantallas principales
- `lib/services/` - Servicios (API, grabación, etc)
- `lib/providers/` - State management con Provider
- `lib/widgets/` - Componentes reutilizables

## Requisitos

- Flutter 3.0+
- Dart 3.0+
- Permisos de micrófono y acceso a contactos
